# Arduino Operation
## LEFT STATUS LED
| LED Operation | System status                |
|---------------|------------------------------|
| Off           | Not connected to WiFi        |
| Slow Blinking | Connecting to Server         |
| On            | Connected to WiFi and Server |

## GPS STATUS LED
| LED Operation | System status                |
|---------------|------------------------------|
| Off           | GPS module not powered       |
| On            | Powered, no Fix              |
| Blinking      | GPS powered with a Fix       |
